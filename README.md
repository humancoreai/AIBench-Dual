# DualAI Workbench (Skeleton)

## Was es ist
Eine Desktop-Workbench mit zwei Fenstern (Pane A / Pane B), die unterschiedliche KI‑Anbieter
über deren eigene Web‑Oberflächen nutzt und lokale Projekte (file‑first) verwaltet.

## Was es nicht ist
- Kein eigener Chatbot, kein Modellhosting
- Keine verpflichtende API‑Integration
- Keine UI‑Automatisierung oder Scraping
- Kein implizites Produktversprechen (Challenger‑Projekt)

## Status
Repo-Skeleton, noch keine Runtime-Implementierung.
